# 历知
历知是一个  macOS 日历/提醒事项 导出工具。 

使用 iOS/macOS 原生的 日历/提醒事项，可以便捷免费的管理日程和计划；

本工具可以导出备份数据，希望对你有帮助！

欢迎贡献代码 💐
